## What this replaces

Drop-in replacement for [`is-fullwidth-code-point`](https://www.npmjs.com/package/is-fullwidth-code-point). Zero dependencies, 21 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
