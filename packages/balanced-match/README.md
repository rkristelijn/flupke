## What this replaces

Drop-in replacement for [`balanced-match`](https://www.npmjs.com/package/balanced-match). Zero dependencies, 27 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
