## What this replaces

Drop-in replacement for [`resolve-from`](https://www.npmjs.com/package/resolve-from). Zero dependencies, 11 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
