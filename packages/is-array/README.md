## Install

```bash
npm i @flupkejs/is-array
```

## What this replaces

Drop-in replacement for [`is-array`](https://www.npmjs.com/package/is-array). Zero dependencies, 2 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
