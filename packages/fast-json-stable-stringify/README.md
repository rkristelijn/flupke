## What this replaces

Drop-in replacement for [`fast-json-stable-stringify`](https://www.npmjs.com/package/fast-json-stable-stringify). Zero dependencies, 34 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
