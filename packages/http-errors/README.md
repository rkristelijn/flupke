## What this replaces

Drop-in replacement for [`http-errors`](https://www.npmjs.com/package/http-errors). Zero dependencies, 94 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
