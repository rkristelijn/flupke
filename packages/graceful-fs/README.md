## What this replaces

Drop-in replacement for [`graceful-fs`](https://www.npmjs.com/package/graceful-fs). Zero dependencies, 32 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
