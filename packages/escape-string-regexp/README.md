## What this replaces

Drop-in replacement for [`escape-string-regexp`](https://www.npmjs.com/package/escape-string-regexp). Zero dependencies, 3 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
