## What this replaces

Drop-in replacement for [`is-path-inside`](https://www.npmjs.com/package/is-path-inside). Zero dependencies, 5 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
