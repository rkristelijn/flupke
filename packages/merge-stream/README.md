## What this replaces

Drop-in replacement for [`merge-stream`](https://www.npmjs.com/package/merge-stream). Zero dependencies, 19 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
