## What this replaces

Drop-in replacement for [`find-up`](https://www.npmjs.com/package/find-up). Zero dependencies, 19 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
