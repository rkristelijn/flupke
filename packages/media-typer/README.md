## What this replaces

Drop-in replacement for [`media-typer`](https://www.npmjs.com/package/media-typer). Zero dependencies, 44 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
