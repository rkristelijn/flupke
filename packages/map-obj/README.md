## What this replaces

Drop-in replacement for [`map-obj`](https://www.npmjs.com/package/map-obj). Zero dependencies, 12 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
