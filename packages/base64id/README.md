## What this replaces

Drop-in replacement for [`base64id`](https://www.npmjs.com/package/base64id). Zero dependencies, 4 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
