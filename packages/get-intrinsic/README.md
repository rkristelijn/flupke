## Install

```bash

```

## What this replaces

Drop-in replacement for [`get-intrinsic`](https://www.npmjs.com/package/get-intrinsic). Zero dependencies, 94 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
