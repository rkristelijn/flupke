## What this replaces

Drop-in replacement for [`encodeurl`](https://www.npmjs.com/package/encodeurl). Zero dependencies, 82 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
