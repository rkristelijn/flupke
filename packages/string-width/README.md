## What this replaces

Drop-in replacement for [`string-width`](https://www.npmjs.com/package/string-width). Zero dependencies, 26 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
