```bash

```

## What this replaces

Drop-in replacement for [`import-fresh`](https://www.npmjs.com/package/import-fresh). Zero dependencies, 9 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
