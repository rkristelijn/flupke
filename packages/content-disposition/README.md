## What this replaces

Drop-in replacement for [`content-disposition`](https://www.npmjs.com/package/content-disposition). Zero dependencies, 68 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
