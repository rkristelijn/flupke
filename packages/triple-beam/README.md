## What this replaces

Drop-in replacement for [`triple-beam`](https://www.npmjs.com/package/triple-beam). Zero dependencies, 76 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
