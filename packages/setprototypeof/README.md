## What this replaces

Drop-in replacement for [`setprototypeof`](https://www.npmjs.com/package/setprototypeof). Zero dependencies, 7 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
