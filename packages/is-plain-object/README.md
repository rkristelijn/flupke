## What this replaces

Drop-in replacement for [`is-plain-object`](https://www.npmjs.com/package/is-plain-object). Zero dependencies, 9 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
