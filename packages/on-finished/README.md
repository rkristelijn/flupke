## What this replaces

Drop-in replacement for [`on-finished`](https://www.npmjs.com/package/on-finished). Zero dependencies, 51 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
