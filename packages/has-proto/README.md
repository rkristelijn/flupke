## What this replaces

Drop-in replacement for [`has-proto`](https://www.npmjs.com/package/has-proto). Zero dependencies, 3 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
