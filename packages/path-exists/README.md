## What this replaces

Drop-in replacement for [`path-exists`](https://www.npmjs.com/package/path-exists). Zero dependencies, 22 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
