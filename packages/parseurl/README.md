## What this replaces

Drop-in replacement for [`parseurl`](https://www.npmjs.com/package/parseurl). Zero dependencies, 46 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
