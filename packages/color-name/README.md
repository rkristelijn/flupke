## Install

```bash
npm i @flupkejs/color-name
```

## What this replaces

Drop-in replacement for [`color-name`](https://www.npmjs.com/package/color-name). Zero dependencies, 150 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
