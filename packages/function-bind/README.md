## Install

```bash
npm i @flupkejs/function-bind
```

## What this replaces

Drop-in replacement for [`function-bind`](https://www.npmjs.com/package/function-bind). Zero dependencies, 2 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
