## What this replaces

Drop-in replacement for [`mimic-fn`](https://www.npmjs.com/package/mimic-fn). Zero dependencies, 7 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
