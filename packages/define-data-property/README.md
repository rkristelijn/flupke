## Install

```bash
npm i @flupkejs/define-data-property
```

## What this replaces

Drop-in replacement for [`define-data-property`](https://www.npmjs.com/package/define-data-property). Zero dependencies, 1 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
