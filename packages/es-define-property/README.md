## What this replaces

Drop-in replacement for [`es-define-property`](https://www.npmjs.com/package/es-define-property). Zero dependencies, 1 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
