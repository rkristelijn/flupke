```bash

```

## What this replaces

Drop-in replacement for [`emoji-regex`](https://www.npmjs.com/package/emoji-regex). Zero dependencies, 3 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
