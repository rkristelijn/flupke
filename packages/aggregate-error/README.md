## What this replaces

Drop-in replacement for [`aggregate-error`](https://www.npmjs.com/package/aggregate-error). Zero dependencies, 12 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
