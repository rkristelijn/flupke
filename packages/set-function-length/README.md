## What this replaces

Drop-in replacement for [`set-function-length`](https://www.npmjs.com/package/set-function-length). Zero dependencies, 9 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
