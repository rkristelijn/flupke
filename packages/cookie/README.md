## What this replaces

Drop-in replacement for [`cookie`](https://www.npmjs.com/package/cookie). Zero dependencies, 39 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
