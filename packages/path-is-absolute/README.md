## What this replaces

Drop-in replacement for [`path-is-absolute`](https://www.npmjs.com/package/path-is-absolute). Zero dependencies, 1 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
