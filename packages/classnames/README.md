## Install

```bash

```

## What this replaces

Drop-in replacement for [`classnames`](https://www.npmjs.com/package/classnames). Zero dependencies, 19 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
