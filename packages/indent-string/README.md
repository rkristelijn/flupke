## What this replaces

Drop-in replacement for [`indent-string`](https://www.npmjs.com/package/indent-string). Zero dependencies, 8 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
