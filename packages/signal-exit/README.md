## What this replaces

Drop-in replacement for [`signal-exit`](https://www.npmjs.com/package/signal-exit). Zero dependencies, 16 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
