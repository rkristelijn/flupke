## What this replaces

Drop-in replacement for [`content-type`](https://www.npmjs.com/package/content-type). Zero dependencies, 64 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
