## What this replaces

Drop-in replacement for [`memoize-one`](https://www.npmjs.com/package/memoize-one). Zero dependencies, 21 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
