## What this replaces

Drop-in replacement for [`merge-descriptors`](https://www.npmjs.com/package/merge-descriptors). Zero dependencies, 19 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
