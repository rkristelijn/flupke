## What this replaces

Drop-in replacement for [`path-type`](https://www.npmjs.com/package/path-type). Zero dependencies, 26 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
