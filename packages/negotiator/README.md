## What this replaces

Drop-in replacement for [`negotiator`](https://www.npmjs.com/package/negotiator). Zero dependencies, 139 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
