## What this replaces

Drop-in replacement for [`toidentifier`](https://www.npmjs.com/package/toidentifier). Zero dependencies, 18 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
