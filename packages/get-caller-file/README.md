## What this replaces

Drop-in replacement for [`get-caller-file`](https://www.npmjs.com/package/get-caller-file). Zero dependencies, 11 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
