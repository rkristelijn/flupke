```bash

```

## What this replaces

Drop-in replacement for [`dotenv-expand`](https://www.npmjs.com/package/dotenv-expand). Zero dependencies, 35 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
