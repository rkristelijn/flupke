
## What this replaces

Drop-in replacement for [`type-detect`](https://www.npmjs.com/package/type-detect). Zero dependencies, 10 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
