## What this replaces

Drop-in replacement for [`get-stream`](https://www.npmjs.com/package/get-stream). Zero dependencies, 13 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
