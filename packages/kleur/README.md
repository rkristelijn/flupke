## What this replaces

Drop-in replacement for [`kleur`](https://www.npmjs.com/package/kleur). Zero dependencies, 62 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
