## What this replaces

Drop-in replacement for [`deep-equal`](https://www.npmjs.com/package/deep-equal). Zero dependencies, 1 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
