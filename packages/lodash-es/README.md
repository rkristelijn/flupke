```bash

```

## What this replaces

Drop-in replacement for [`lodash-es`](https://www.npmjs.com/package/lodash-es). Zero dependencies, 71 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
