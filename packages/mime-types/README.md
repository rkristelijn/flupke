```bash

```

## What this replaces

Drop-in replacement for [`mime-types`](https://www.npmjs.com/package/mime-types). Zero dependencies, 299 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
