## What this replaces

Drop-in replacement for [`micromatch`](https://www.npmjs.com/package/micromatch). Zero dependencies, 74 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
