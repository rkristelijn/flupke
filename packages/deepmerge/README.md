## What this replaces

Drop-in replacement for [`deepmerge`](https://www.npmjs.com/package/deepmerge). Zero dependencies, 34 lines of code.

Only the API surface used by major frameworks (Express, Next.js, Jest, etc.) is implemented — unused code is stripped.
